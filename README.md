# Entorno de desarrollo para React.js

## Sección I - Inicialización
Inicializamos un repositorio de ***git*** y un paquete con ***npm***.
También creamos los primeros archivos y estructura de carpetas.

- npm init -y
- git init
- Crear carpetas
   1. /src
   2. /public
   3. /src/components 
   4. /src/containers
- Crear archivos
   1. /src/index.js
   2. /public/index.html
